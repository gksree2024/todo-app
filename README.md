# 📝 Todo App

A simple and responsive Todo App built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

🔗 https://gksree2024.github.io/todo-app/

---

## 📌 Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Tasks saved using Local Storage
* Responsive and clean UI
* Beginner-friendly project
* Edit task feature
* Dark/Light mode
* Task categories
* Due dates

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub
* GitHub Pages
